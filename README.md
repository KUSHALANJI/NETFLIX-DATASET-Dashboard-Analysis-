# Netflix Dataset Analysis Dashboard 📊

## Project Overview
This project focuses on analyzing Netflix movies and TV shows data to discover meaningful insights and trends. The project follows an end-to-end data analysis workflow starting from raw dataset collection, data cleaning, exploratory data analysis, visualization, and dashboard creation.

## Objectives
- Analyze Netflix content distribution
- Identify trends in movies and TV shows
- Explore genres, ratings, countries, and release patterns
- Create interactive dashboards for better understanding of data

## Dataset
Source: Kaggle Netflix Dataset

The dataset contains information about:
- Title
- Type (Movie/TV Show)
- Director
- Cast
- Country
- Release Year
- Rating
- Duration
- Genre
- Date Added

## Tools & Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI / Dashboard Tool

## Project Workflow

### 1. Data Collection
- Downloaded the raw Netflix dataset from Kaggle
- Imported the dataset into Jupyter Notebook using Pandas

### 2. Data Cleaning
Performed preprocessing steps:
- Removed duplicate records
- Handled missing values
- Converted data types
- Cleaned inconsistent data
- Extracted useful features from existing columns

### 3. Exploratory Data Analysis (EDA)
Analyzed the dataset to find:
- Number of movies and TV shows
- Content growth over years
- Popular genres
- Country-wise content distribution
- Rating analysis
- Release trends

### 4. Data Analysis
Used:
- Pandas for data manipulation
- NumPy for statistical calculations

Performed:
- Aggregations
- Grouping
- Trend analysis
- Comparative analysis
